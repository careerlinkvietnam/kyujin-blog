# バクニン省の工業団地ガイド【2026年最新版】

**最終更新：2026年1月**

バクニン省はサムスンを中心とした電子機器産業の集積地で、2024年にはベトナム最大の52.1億ドルのFDIを誘致しました。ハノイから30kmという好立地で、北部電子産業のハブとなっています。

**→ [ベトナム工業団地ガイド（総論）に戻る](https://kyujin.careerlink.asia/blog/industrial-park-vietnam)**

---

## 基本データ（2025年）

| 項目 | 数値 |
|------|------|
| 工業団地数 | **32〜33ヵ所** |
| 計画面積 | 10,100ヘクタール以上 |
| 稼働中面積 | 6,397ヘクタール（16ヵ所） |
| 入居率 | 61〜86%（タイプによる） |
| 2024年FDI誘致 | **52.1億ドル**（ベトナム1位） |

---

## サムスンエコシステム

| 項目 | 内容 |
|------|------|
| サムスンのベトナム総投資額 | 232億ドル |
| バクニン集中度 | 約50% |
| 主要施設 | イェンフォンのディスプレイ・部品工場（18億ドル） |
| 生産能力 | 月産1,100万台のモバイルデバイス |
| 関連企業 | 100社以上のサプライヤー |

サムスンは2008年から操業を開始し、バクニン省の電子機器産業を牽引しています。
