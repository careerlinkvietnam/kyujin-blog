## 1. 私たちは何者か（最重要）

### CareerLink Asiaは人材紹介会社である

**核心事業**: 東南アジア（特にベトナム・タイ）にある日系企業が人材を探す際に、現地人材および日本人材を紹介すること

### 顧客の優先順位

| 優先度 | 顧客タイプ | 説明 |
|-------|-----------|------|
| **1位** | **日系企業（採用側）** | ベトナム・タイで人材を探している日系企業 |
| 2位 | 求職者（日本人） | 東南アジアで働きたい日本人 |
| 3位 | 求職者（現地人材） | 日系企業で働きたい現地人材 |

**B2B（企業顧客へのリーチ）が最も重要。求職者向けコンテンツは人材プールを構築するための手段。**

### 対象地域

| 項目 | 内容 |
|------|------|
| **対象地域** | **タイ、ベトナム**（2カ国に特化） |
| **サービス** | 東南アジアへの転職支援 |
| **特徴** | 現地在住の日本人コンサルタントがサポート |

**注意**: シンガポール、マレーシア、インドネシアは対象外。「東南アジア全域」「ASEAN各国」などの表現は使用しない。

### 絶対にやってはいけないこと

| # | 禁止事項 | 理由 |
|---|---------|------|
| 1 | **「法人設立」サービスを提案する** | 私たちは人材紹介会社。法人設立サービスは提供していない |
| 2 | **「投資アドバイス」を提供する** | 投資コンサルタントではない |
| 3 | **政府・政策を批判する** | 現地での事業継続リスク、サイトの信頼性低下 |
| 4 | **架空のデータを捏造する** | 「n=500」「CareerLink Asia調べ」など偽造禁止 |
| 5 | **手動目次を追加する** | WordPressプラグインと重複する |
| 6 | **日本語をURLスラッグにする** | SEOに悪影響、エンコードで長くなる |

## 5. 記事品質基準 - 東南アジア最高の情報サイトを目指して

<div style="background: #e8f5e9; border: 3px solid #2e7d32; border-radius: 8px; padding: 20px; margin: 20px 0;">

**目標**: 東南アジアで生活・仕事をする日本人にとって「このサイトを見れば間違いない」と言われる情報サイトになること。

</div>

### 品質の5原則

| # | 原則 | 基準 | 確認方法 |
|---|------|------|---------|
| 1 | **正確性** | 情報が正しく、検証可能 | 一次情報源へのリンク確認 |
| 2 | **鮮度** | 情報が最新である | 法改正・データ更新日の確認 |
| 3 | **網羅性** | 読者の疑問をすべてカバー | 競合記事との比較 |
| 4 | **明瞭性** | 競合より分かりやすい | 構成・表現の比較 |
| 5 | **実用性** | 読者が行動できる | 具体的手順・連絡先の有無 |

### 品質チェックリスト（記事公開・確認時に必須）

#### A. 一次情報チェック ✅

| チェック項目 | 確認内容 |
|-------------|---------|
| □ 法律・規制 | 政府公式サイト（労働省、移民局等）を参照したか |
| □ 統計データ | 公式統計局のデータを使用したか |

| □ 言語 | 現地語（タイ語/ベトナム語）の一次情報を確認したか |
| □ 出典明記 | 重要な数字・事実に出典を明記したか |

**使用OK**: 政府機関、JETRO、大使館、公式統計局
**使用NG**: コンサル会社、法律事務所コラム、ニュースサイトの解説

#### B. 競合比較チェック ✅

| チェック項目 | 確認内容 |
|-------------|---------|
| □ 情報量 | 競合上位3サイトより情報が多いか |
| □ 分かりやすさ | 競合より構成が整理されているか |
| □ 具体性 | 競合にない具体的情報（数字、手順、連絡先）があるか |
| □ 独自価値 | 「このサイトでしか得られない」情報があるか |
| □ 更新日 | 競合より情報が新しいか |

**競合分析の実施**:


## 7. 情報の正確性（最重要セクション）

<div style="background: #fff8e1; border: 3px solid #f57c00; border-radius: 8px; padding: 20px; margin: 20px 0;">

### 🔑 日本一のサイトを作る鍵：情報源の優先順位

**日本語の情報だけを参照していては、絶対に日本一のサイトにはなれない。**

競合他社と同じ日本語情報を参照していては、同じ品質の記事しか作れない。
**一次情報**と**現地語情報**を参照することで初めて、競合を超える記事が作れる。

</div>

### 情報源の優先順位（必ず守ること）

| 優先度 | 情報源 | 説明 | 使い方 |
|-------|-------|------|-------|
| **1位** | **一次情報（政府・公的機関）** | 法律原文、政府発表、公式統計、条文 | **記事の根拠として必須**。断定表現OK |
| **2位** | **現地語情報（タイ語・ベトナム語）** | 現地政府サイト、現地メディア、現地専門家の解説 | **日本語にない最新・詳細情報を取得** |
| **3位** | 日本語情報（JETRO、大使館等） | 日本機関の発表、日本語メディア | **参考程度のみ**（これだけでは不十分） |

<div style="background: #ffebee; border: 2px solid #c62828; border-radius: 8px; padding: 15px; margin: 15px 0;">

**❌ 禁止**: 日本語の情報だけを参照して記事を作成すること

**理由**:
- 日本語情報は二次情報であり、遅れ・誤り・省略がある
- 競合他社も同じ日本語情報を見ている（差別化できない）
- 現地語でしか得られない詳細・最新情報がある

</div>

### 国別の一次情報源

| 国 | 参照すべき言語 | 一次情報源の例 |
|----|--------------|---------------|
| ベトナム | **ベトナム語** | 政府公式サイト、現地メディア、労働法原文、Thông tư（通達） |
| タイ | **タイ語** | BOI、労働省、Royal Gazette（官報）、現地メディア |

**注意**: JETRO・大使館は「使用OK」だが、これらは**参考情報**として使う。記事の根拠は現地語の一次情報から取ること。

- 次回作業時に同じ確認作業を繰り返さない（手戻り防止）

- 他の作業者（または将来の自分）が状況を把握できる

| 確認内容 | 更新先 |

## 17. よく使うコマンド・スクリプトパターン

### WordPress API操作（PowerShell）

```powershell
# 認証ヘッダー
$user = "careerlinkasia"
$pass = "N2Zz bzSn AWve Pa83 Ap2S 6Mlw"
$base64Auth = [Convert]::ToBase64String([Text.Encoding]::ASCII.GetBytes("${user}:${pass}"))
$headers = @{
    Authorization = "Basic $base64Auth"
    "Content-Type" = "application/json; charset=utf-8"
}

# 記事取得
$article = Invoke-RestMethod -Uri "https://kyujin.careerlink.asia/blog/wp-json/wp/v2/posts/[ID]" -Headers $headers

# 記事更新
$body = @{ content = $newContent } | ConvertTo-Json -Compress
Invoke-RestMethod -Uri "https://kyujin.careerlink.asia/blog/wp-json/wp/v2/posts/[ID]" -Headers $headers -Method Post -Body ([System.Text.Encoding]::UTF8.GetBytes($body))
```
