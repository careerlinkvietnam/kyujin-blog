## 作業開始時の手順

1. **`rule/00_必読ルール.md` を読む**
2. **`記事インデックス.md` でタスク・進捗を確認**
3. ユーザーの指示を聞く
4. 該当するマニュアルを読む（rule/配下）
5. 作業を実行
6. `記事インデックス.md` を更新
