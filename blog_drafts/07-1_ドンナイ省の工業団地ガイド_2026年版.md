# ドンナイ省の工業団地ガイド【2026年最新版】

**最終更新：2026年1月**

ドンナイ省は南部最大の工業団地集積地で、59ヵ所の工業団地が立地しています。2025年末に開港したロンタイン国際空港に隣接し、輸出製造業にとって最適な立地条件を備えています。

**→ [ベトナム工業団地ガイド（総論）に戻る](https://kyujin.careerlink.asia/blog/industrial-park-vietnam)**

---

## 基本データ（2025年）

| 項目 | 数値 |
|------|------|
| 工業団地数 | **59ヵ所**（マスタープラン81ヵ所中） |
| 総面積 | 約21,700ヘクタール |
| 稼働中 | 42ヵ所 |
| 平均入居率 | **約95%** |
| 日系投資額 | 52.2億ドル（276プロジェクト） |

---

## ロンタイン国際空港の開港

| 項目 | 詳細 |
|------|------|
| 技術運用開始 | 2025年12月19日（実施済み） |
| 商業運用開始 | **2026年6月予定** |
| 年間旅客能力 | 2,500万人（第1期） |
| 距離 | ロンタイン工業団地から直近 |

ホーチミン発着の全国際線が段階的に移管予定。精密機器・電子部品など時間的制約のある輸出に最適。

---

## 主要工業団地

### 1. アマタ工業団地（ビエンホア）

| 項目 | 内容 |
|------|------|
| 面積 | 513ヘクタール |
| 入居率 | **90%超** |
| 開発企業 | AMATA Corporation（タイ） |
| 主な日系テナント | NOK、東京インキ、岡本工業 |

精密機械・電子部品・物流に強み。アマタロジスティクスセンターがASEAN・中国向け物流拠点として機能。

### 2. ニョンチャック工業団地群

| 項目 | 内容 |
|------|------|
| 工業団地数 | 9ヵ所 |
| 投資プロジェクト | 624件（FDI 451件、約113億ドル） |
| 主な日系企業 | YKK |

ベトナム最大の工業団地集積地。フックアン港（年間500万TEU）に隣接。

### 3. ロンドゥック工業団地（双日）

| 項目 | 内容 |
|------|------|
| 面積 | 330ヘクタール |
| 開発企業 | **双日** |
| ロングドゥック3 | 244.5ha（2025年12月着工） |

日系企業による開発。ロンタイン空港に隣接し、航空輸出に最適な立地。

### 4. 新規開発中の大型工業団地

**バウカン・タンヒエップ工業団地**
- 面積：1,000ヘクタール
- 投資額：約4.4億ドル
- 特徴：次世代型工業都市モデル

**スアンクエ・ソンニャン工業団地**
- 面積：1,000ヘクタール
- 着工：2025年12月

## 2024〜2025年のトピックス

### 半導体産業の進出

- **Coherent（米国）**：ニョンチャック1工業団地に1.27億ドルの半導体工場を開設（2025年7月）
- 追加投資：10億ドル以上を計画

### ロンタイン国際空港開港

- 商業運用：2026年6月予定
- 全国際線のホーチミンからの移管により、周辺工業団地の価値が大幅上昇

### FDI誘致実績

- 2024年：16億ドル
- 2025年1〜9月：14億ドル（全国トップ5）

## こんな企業におすすめ

- 航空輸出が多い電子部品・精密機器メーカー
- ASEAN向け製造・輸出拠点を検討中の企業
- 既存の日系サプライチェーンに参入したい企業
- ホーチミン経済圏でのビジネス拡大を目指す企業
