**このマニュアルは、すべての記事作成・リライト作業において遵守すること。**

## 目次について（重要）

### 手動目次は不要

WordPressには目次自動生成プラグインがインストールされているため、**記事内に手動で目次を追加する必要はない**。

### NG例

```html
<!-- これは追加しないでください -->
<h2>目次</h2>
<ul><li>セクション1</li></ul>
```

### OK例

見出し（h2, h3）を適切に使用すれば、プラグインが自動的に目次を生成する。

## 記事構成の基本

### 推奨構成

1. **導入文** - 記事の概要、対象読者、読むメリット
2. **本記事のポイント（要約）** - 箇条書きで主要ポイント
3. **本文** - h2/h3で構造化
4. **FAQ** - よくある質問
5. **まとめ** - 結論、CTA
6. **関連記事** - 内部リンク

### 見出しレベル

- `<h2>` : メインセクション（目次に表示される）
- `<h3>` : サブセクション（目次に表示される）
- `<h4>` : 小見出し（目次に表示されない場合あり）

## 対象読者（ターゲット）の明確化

### 対象読者の分類

| 分類 | 説明 | タイトル例 |
|-----|------|-----------|
| **求職者** | 転職・就職希望者 | 「〇〇転職ガイド」「〇〇で働く」 |
| **企業・経営者** | 進出企業、投資家 | 「〇〇投資ガイド」「外資規制」 |
| **企業・人事** | 人事・採用担当者 | 「労働法ガイド（経営者向け）」 |
| **全般** | 特定しない | 「最低賃金」「経済動向」 |

### タイトルの付け方

**求職者向け：ポジティブな表現**
- ✅ 「タイ就労ビザ・ワークパーミット完全ガイド」
- ❌ 「タイ外国人雇用規制ガイド」（堅苦しい）

**企業向け：専門的でも可**
- ✅ 「タイ労働法ガイド【経営者・人事向け】」
- ✅ 「ベトナム外資規制・投資法ガイド」

## 競合分析（必須プロセス）

### 競合分析の手順

#### Step 1: 検索キーワードの特定

記事のターゲットキーワードを明確にする。

#### Step 2: 競合ページの調査

Google検索で上位5〜10位のページを確認し、以下を分析：

| 分析項目 | 確認内容 |
|---------|---------|
| 情報量 | 文字数、見出し数、紹介している項目数 |
| 具体性 | 具体的な数字、名前、住所、料金の有無 |
| 網羅性 | カバーしているトピックの範囲 |
| 更新日 | 情報の新しさ |
| 独自情報 | 他にはない情報の有無 |

#### Step 3: 競合分析シートの作成

```markdown
## 競合分析シート

### ターゲットキーワード: [キーワード]

### 競合ページ一覧

| 順位 | サイト名 | URL | 文字数 | 紹介数 | 強み | 弱み |
|-----|---------|-----|-------|-------|------|------|
| 1 | | | | | | |
| 2 | | | | | | |

### 競合に勝つために必要なこと
1.
2.

### 当サイト記事の差別化ポイント
1.
2.
```

## 品質の低い記事の特徴（これを避ける）

- 汎用的な説明文が多い（「〜が重要です」「〜を検討しましょう」）
- 具体的な数字、名前、住所、料金がない
- どのサイトにも書いてある情報しかない
- 読んでも何をすればいいかわからない
- 情報が古い、または検証されていない

### 特に注意すべきパターン

| パターン | なぜ危険か | 対策 |
|---------|----------|------|
| 「〜が重要です」で終わる文 | 具体性がない | 具体的な方法・例を追加 |
| 「〜を検討しましょう」 | 行動に繋がらない | 具体的な手順を記載 |
| 「さまざまな〜があります」 | 情報量ゼロ | 具体例を列挙 |
| 一般論の羅列 | 独自性がない | 独自の視点・情報を追加 |

## 日本人読者視点の徹底

### 日本人視点で書くべき内容

| 観点 | 日本人が気になること | 記載すべき情報 |
|------|---------------------|----------------|
| **比較** | 日本との違いは？ | 日本の制度・文化との比較 |
| **安全性** | 安全に暮らせる？ | 治安、医療、日本人コミュニティ |
| **言語** | 日本語は通じる？ | 日本語の通用度、英語の必要性 |
| **食事** | 日本食は食べられる？ | 日系スーパー、日本食レストラン情報 |
| **コスト** | 日本円で換算すると？ | 日本との物価比較、為替考慮 |

### 専門用語の日本語化

| 避けるべき表現 | 使うべき表現 |
|--------------|-------------|
| アポスティーユ | **公文書の認証**（アポスティーユ） |
| ワークパーミット | **労働許可証** |
| テンポラリーレジデンスカード | **一時滞在許可証** |

## 出典の記載方法

記事末尾に「情報の出典」セクションを設け、以下の形式で記載：

```html
<p style="font-size: 12px; color: #666; margin-top: 40px; border-top: 1px solid #ddd; padding-top: 16px;">
<strong>情報の出典:</strong><br>
・タイ労働保護法（2025年改正）: <a href="https://...">Tilleke & Gibbins</a><br>
・ベトナム労働法: <a href="https://...">Vietnam.vn</a><br>
<br>
※記事の情報は20XX年X月時点のものです。最新情報は各国の公式サイトをご確認ください。
</p>
```

## 記事タイプ別の品質基準

### 転職・人材紹介会社紹介記事

| 項目 | 最低基準 |
|------|---------|
| 紹介企業数 | 5社以上 |
| 各社の情報 | 会社名、特徴、対象業種、連絡先 |
| 比較表 | あり |
| 選び方ガイド | あり |

### 生活情報記事

| 項目 | 最低基準 |
|------|---------|
| 具体的な場所/店舗 | 3つ以上 |
| 住所・アクセス | 必須 |
| 料金情報 | 可能な限り |
| 実用的なアドバイス | 必須 |

### ビザ・手続き記事

| 項目 | 最低基準 |
|------|---------|
| 申請手順 | ステップバイステップ |
| 必要書類 | 完全なリスト |
| 費用 | 公式情報 |
| 注意点 | 実際の経験に基づく |
| 公式情報源 | リンク必須 |

## 駐在員と現地採用の違いを明記する

海外で働く日本人には「駐在員」と「現地採用」の2種類がある。制度の適用が異なる場合は必ず明記すること。

| 項目 | 駐在員 | 現地採用 |
|------|--------|----------|
| 雇用契約 | 日本本社と契約 | 現地法人と契約 |
| 社会保険 | 日本の社会保険継続 | 現地の社会保険 |
| 産休・育休 | 日本の制度を利用可能 | 現地の制度を利用 |
| 給与体系 | 日本本社基準 | 現地法人基準 |

## カテゴリー割り当て（必須）

**すべての記事公開時に適切なカテゴリーを設定すること。**

### チェックリスト

- [ ] 特定の国に関する記事 → 国カテゴリーを設定（タイ:26, ベトナム:25 など）
- [ ] 複数国を扱う記事 → すべての国カテゴリーを設定
- [ ] 転職ノウハウ系（国を問わない）→ ビジネス(148)を設定
- [ ] 労働法・ビザ関連 → 国 + ビジネス(148) + 法律(158)
- [ ] 生活情報 → 国 + 生活情報(28)

### 詳細ルール

詳細は `03_カテゴリー割り当て.md` を参照。

### WordPress API でのカテゴリー設定

新規投稿時は `categories` フィールドを必ず含める：

```javascript
const postData = JSON.stringify({
    title: '記事タイトル',
    content: content,
    status: 'publish',
    slug: 'article-slug',
    categories: [26, 148]  // 必ず設定すること
});
```
