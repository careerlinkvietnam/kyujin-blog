# ハイフォン市の工業団地ガイド【2026年最新版】

**最終更新：2026年1月**

ハイフォン市は北部最大の港湾都市で、ラックフエン深水港を擁します。日系ではノムラ工業団地（1994年設立、北部初のFDI工業団地）や双日が開発するノムディン・ハイフォン工業団地があり、港湾物流に強みを持つ製造拠点として発展しています。

**→ [ベトナム工業団地ガイド（総論）に戻る](https://kyujin.careerlink.asia/blog/industrial-park-vietnam)**

---

## 基本データ（2025年）

| 項目 | 数値 |
|------|------|
| 工業団地数 | **40〜43ヵ所** |
| 総面積 | 11,208〜12,100ヘクタール |
| 2025年新規設立 | 8ヵ所 |
| 累計FDI | 422億ドル（1,005プロジェクト） |
| 日系投資 | 39億ドル（146プロジェクト、第2位） |

---

## ラックフエン深水港の優位性

| 項目 | 内容 |
|------|------|
| 対応船舶 | **16.5万DWT**、12,000TEU |
| 岸壁長 | 2.4km |
| 水深 | 16.8〜18.4m |
| 直航路 | 欧米への直行ルート（シンガポール・香港経由不要） |

**メリット**
- 物流コスト・時間の大幅削減
- 18,000TEU以上の大型コンテナ船が直接寄港可能
- ベトナムで2つしかない深水港のひとつ

## 主要工業団地

### 1. DEEP C工業団地群

| 項目 | 内容 |
|------|------|
| 面積 | 3,400ヘクタール（ハイフォン・クアンニンで5ゾーン） |
| 設立 | 1997年 |
| 投資額 | 35億ドル（120プロジェクト以上） |
| 主なテナント | ブリヂストン、出光、Knauf、Chevron、tesa |

石油化学エリアに液体貨物用埠頭システムを保有。エコ工業団地コンセプトで再生エネルギー・リサイクル素材道路を推進。

### 2. VSIP ハイフォン

| 項目 | 内容 |
|------|------|
| 面積 | 1,600ヘクタール（工業500ha＋都市開発1,100ha） |
| 投資額 | 10億ドル |
| 入居率 | **76%以上** |
| 雇用 | 3万人以上 |

**日系投資企業**
- **京セラ**：4.25億ドル（事務機器製造）
- **ニプロ**：2.5億ドル（製薬・医療機器）
- **富士フイルム**：プリンター・MFP製造

### 3. ノムラ工業団地（日本工業団地ハイフォン）

| 項目 | 内容 |
|------|------|
| 設立 | **1994年12月23日**（北部初のFDI工業団地） |
| 開発 | ハイフォン市＋野村グループ（日本） |
| 面積 | 353ヘクタール（Phase 1: 153ha、Phase 2: 200ha） |
| FDI誘致額 | 約10億ドル |

**主な日系テナント**
- 矢崎総業（自動車用ワイヤーハーネス）
- トヨタ紡織（自動車部品）
- 豊田合成（ゴム・プラスチック製品）
- パイオニア（電子機器）

### 4. ノムディン・ハイフォン工業団地（双日）

| 項目 | 内容 |
|------|------|
| 面積 | 300ヘクタール |
| 開発 | **双日** |
| 特徴 | 日系開発による最新インフラ |

## 日系企業・開発会社の存在感

### 双日

- ノムディン・ハイフォン工業団地（300ha）の開発
- DEEP C工業団地・Bamboo Capitalとの協力MoU（2021年11月）
- 工業団地＋屋上太陽光発電のインフラサービス
- 推定投資額：約2.5億ドル

### 住友電装（スミデンソーベトナム）

| 項目 | 内容 |
|------|------|
| 事業内容 | ワイヤーハーネス組立（ベトナム最大級） |
| 位置 | ダイアン工業団地、トゥーミン区 |
| 従業員 | 約12,000人 |

### 日系企業の集中度

- ベトナムの日系企業の**42%**が紅河デルタ地域（ハイフォン含む）に集中
- 日本はハイフォンのFDI第2位（韓国に次ぐ）
