## 1. 現状分析

### 1.1 サイト概要
- **事業内容:** タイ・ベトナムの日系企業向け人材紹介
- **ターゲット:**
  - 日本人求職者（現地就職希望者）
  - 日系企業（現地採用希望）
  - 現地人材（日系企業就職希望者）
- **コンテンツ:** 求人情報 + ブログ「暮らすアジア」

### 1.2 現在の強み
- 80以上のブログ記事
- 3言語対応（日本語・英語・ベトナム語）
- 給与計算ツール（vietnam-salary-nextjs）
- 現地オフィス（ベトナム・タイ）

### 1.3 現在の課題
- 大手競合との差別化不足
- SEO最適化の余地あり
- コンテンツの重複・類似記事
- E-E-A-T（専門性・権威性・信頼性）の不足

### 1.4 主要競合
| サイト名 | 運営 | 強み |
|---------|------|------|
| カモメアジア転職 | リクルート | ブランド力、求人数 |
| RGF | リクルートG | アジア17都市ネットワーク |
| R-VIETNAM | G.A.コンサルタンツ | ベトナム特化、現地密着 |
| THAI SCOUT | - | タイ特化、スカウト型 |
| JAC Recruitment | JAC | グローバルネットワーク |

## 2. SEO改善計画

### 2.1 技術的SEO（Technical SEO）

#### 即座に実施すべき項目
- [ ] **sitemap.xml の作成・送信**
  - 求人ページ、ブログ記事すべてを含める
  - Google Search Console に送信

- [ ] **robots.txt の最適化**
  ```
  User-agent: *
  Allow: /
  Sitemap: https://kyujin.careerlink.asia/sitemap.xml
  ```

- [ ] **構造化データ（Schema.org）の実装**
  - `JobPosting` スキーマ（求人ページ）
  - `Article` スキーマ（ブログ記事）
  - `Organization` スキーマ（会社情報）
  - `BreadcrumbList` スキーマ（パンくず）

- [ ] **Core Web Vitals の改善**
  - LCP（最大コンテンツ描画）: 2.5秒以内
  - FID（初回入力遅延）: 100ms以内
  - CLS（累積レイアウトシフト）: 0.1以内

- [ ] **hreflang タグの実装**
  ```html
  <link rel="alternate" hreflang="ja" href="https://kyujin.careerlink.asia/ja/" />
  <link rel="alternate" hreflang="en" href="https://kyujin.careerlink.asia/en/" />
  <link rel="alternate" hreflang="vi" href="https://kyujin.careerlink.asia/vi/" />
  ```

## 5. マーケティング戦略

### 5.1 SNSマーケティング

#### プラットフォーム別戦略
| SNS | ターゲット | コンテンツ | 投稿頻度 |
|-----|-----------|-----------|---------|
| Twitter/X | 転職検討者 | 求人情報、業界ニュース | 毎日 |
| Facebook | 30-40代転職者 | 生活情報、イベント告知 | 週3回 |
| LinkedIn | ビジネスパーソン | 業界レポート、求人 | 週2回 |
| Instagram | 若年層 | 現地の魅力、オフィス紹介 | 週3回 |
| YouTube | 全年齢 | インタビュー、現地ガイド | 月2回 |

### 5.2 メールマーケティング

#### メールシリーズ
1. **ウェルカムシリーズ**（登録後7日間）
   - Day 1: 登録お礼＆サービス紹介
   - Day 3: 人気求人紹介
   - Day 5: 転職成功事例
   - Day 7: 個別相談のご案内

2. **週刊ニュースレター**
   - 新着求人ハイライト
   - ブログ記事紹介
   - 現地ニュースピックアップ

3. **セグメント別メール**
   - 国別（ベトナム/タイ）
   - 職種別（IT/営業/製造）
   - 経験年数別

### 5.3 パートナーシップ

#### 提携候補
- 日本語学校（ベトナム・タイ）
- 日本人会・商工会議所
- 大学キャリアセンター
- 転職関連メディア
- インフルエンサー（海外就職系YouTuber等）

### 5.4 オフラインマーケティング

#### イベント企画
- 月1回: オンライン転職相談会
- 四半期: 現地就職説明会（東京・大阪）
- 年2回: 海外就職フェア出展
