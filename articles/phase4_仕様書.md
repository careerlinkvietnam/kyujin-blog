# 【最重要】判断品質の必須要件（全社共通）

## A. 各社紹介で必須の3要素

### 1) 向いている企業タイプ（必須）

各社紹介において、以下のような具体的な記載が必須。

**記載例（OK）**:
- 初進出の日系企業で、現地採用が初めての場合
- 製造業で工場ワーカー・現場管理者の採用が多い企業
- 管理職・日本語話者を探している企業
- 急成長フェーズで大量採用が必要な企業
- ITエンジニア専門で採用したい企業

**NG例**:
- ×「幅広い企業に対応」
- ×「様々なニーズに応える」

### 2) 向かない可能性があるケース（必須）

各社紹介において、以下のような具体的な記載が必須。

**記載例（OK）**:
- 少人数（1-2名）・短期での採用を希望する場合
- 低コスト最優先で紹介手数料を抑えたい場合
- 特殊職種（医療・法務等）の専門人材を探している場合
- 日本語対応が必須だが、現地スタッフのみの対応になる場合
- 地方都市（工業団地等）での採用が中心の場合

**NG例**:
- ×「特にありません」
- ×「ケースバイケース」

### 3) 人事担当者が注意すべきポイント（必須）

各社紹介において、以下の観点から具体的な記載が必須。

**記載すべき観点**:
- 契約形態の注意（成功報酬/前払い/保証期間等）
- スピード感の期待値（初回紹介まで○日目安等）
- コミュニケーション上の特徴（日本人対応可否/レスポンス速度等）
- 料金体系の特徴（最低紹介料/年収比率等）

**記載例（OK）**:
- 「成功報酬型だが、採用決定後30日以内の退職は返金対象外」
- 「初回候補者紹介まで平均2週間、急募案件は要相談」
- 「日本人コンサルタント常駐、日本語でのやり取り可能」
- 「紹介手数料は年収の25%、最低30万バーツから」

**NG例**:
- ×「詳細はお問い合わせください」
- ×「柔軟に対応」

## B. 曖昧表現の禁止（厳守）

以下の表現は**全面禁止**。必ず判断に使える形に言い換える。

| 禁止表現 | 言い換え例 |
|---------|-----------|
| 幅広く対応 | 製造・商社・サービス業の実績あり |
| 実績多数 | 年間200件以上の成約実績（2024年） |
| 豊富な経験 | 創業15年、累計5000名以上の紹介実績 |
| 迅速な対応 | 初回候補者紹介まで平均5営業日 |
| 丁寧なサポート | 入社後3ヶ月間のフォロー面談実施 |
| 高品質なサービス | 入社後6ヶ月定着率92% |

## C. 情報密度の最低基準

| 項目 | 基準 | 備考 |
|------|------|------|
| 1社あたり本文 | **300字以上** | 特徴ボックス・アクセス情報除く |
| 特徴ボックス | **3項目以上** | 全て行動・判断に結びつく文言 |
| 抽象語 | **禁止** | 特徴ボックス内は特に厳格 |

### 特徴ボックスのOK/NG例

**OK例**:
```
- 日本人コンサルタント5名常駐、日本語での対応可能
- 製造業特化、工場長・生産管理職の紹介実績100名以上
- 成功報酬型、入社後60日以内退職は全額返金
```

**NG例**:
```
- 豊富な実績と経験
- きめ細やかなサポート
- 幅広いネットワーク
```

## D. 比較軸の固定（本文中で最低2軸言及必須）

冒頭の4軸（会社規模/拠点/料金/得意分野）に加え、**以下から最低2軸を本文中で言及**すること。

| 比較軸 | 記載例 |
|--------|--------|
| 採用スピード感 | 「初回紹介まで平均1週間」「急募案件対応可」 |
| 得意な採用フェーズ | 「立ち上げ期の少人数採用に強い」「拡大期の大量採用対応可」 |
| 日本語対応レベル | 「日本人常駐」「日本語可能な現地スタッフ対応」「窓口のみ日本語」 |
| 現地拠点の厚み | 「バンコク本社+チョンブリ支店」「ベトナム全土5拠点」 |

## E. レビュー基準（本文生成後の人間レビュー）

本文生成後、人間レビューでは以下**2点のみ**を確認する。

### チェック1: 判断可能性
> この1社紹介だけを読んで「使う / 使わない」の判断ができるか？

**合格基準**:
- 向いている企業タイプが明確
- 向かないケースが明確
- 注意点が具体的

### チェック2: 情報充足性
> 社内の上司・同僚に回しても「情報が薄い」と言われないか？

**合格基準**:
- 料金体系の目安がある
- スピード感の目安がある
- 連絡先・問い合わせ方法が明確

**上記2点を満たさない場合、差し戻し。**

# 共通事項

## v2024版の強み（維持必須）

1. **網羅性**: 市場の主要プレイヤーを網羅
2. **実用性**: アクセス情報・CTAで即問い合わせ可能
3. **比較容易性**: 特徴ボックスで一目で判断可能
4. **判断軸提示**: 冒頭で4軸を明示
5. **信頼性**: 各社の詳細情報（住所・電話・Web）完備

## v2026版で強化すべき点（v2024版の弱み）

1. **判断可能性**: 「向いている企業」「向かないケース」の明示がない
2. **注意喚起**: 人事担当者への具体的注意点がない
3. **情報密度**: 一部の企業で抽象的な表現が残っている
4. **比較軸**: 冒頭4軸以外の比較軸が本文中で不足

## v2026版で絶対にやってはいけないこと

### 構成面
1. 企業数を減らす
2. アクセス情報を省略する
3. 特徴ボックスを削除する
4. CTAボタンを削除する
5. 4軸比較ポイントを省略する

### 表現面
6. 抽象的な説明に置き換える
7. 「幅広く対応」「実績多数」等の曖昧表現を使う
8. 「向いている企業タイプ」を省略する
9. 「向かないケース」を省略する
10. 「注意すべきポイント」を省略する

### 密度面
11. 1社あたり300字未満にする
12. 特徴ボックスを2項目以下にする
13. 追加比較軸の言及を省略する
