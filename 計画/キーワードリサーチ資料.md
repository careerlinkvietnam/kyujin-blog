## 検索意図別キーワード分類

### 情報収集型（Know）
ユーザーは情報を求めている
```
・海外転職 メリット
・タイ 生活費
・ベトナム 治安
・海外転職 必要なスキル
```
**対応:** ブログ記事で詳しく解説

### 比較検討型（Compare）
ユーザーは比較検討している
```
・海外転職エージェント 比較
・タイ ベトナム どっち
・駐在 現地採用 どちら
```
**対応:** 比較記事、ランキング記事

### 行動型（Do）
ユーザーはアクションを取りたい
```
・タイ 求人 応募
・海外転職 エージェント 登録
・ベトナム 求人 検索
```
**対応:** 求人ページ、登録ページへ誘導

### ナビゲーション型（Go）
ユーザーは特定サイトに行きたい
```
・キャリアリンクアジア
・暮らすアジア ブログ
```
**対応:** ブランド認知向上で増やす

## 競合分析

### 主要競合サイト

1. **海外転職サイトA**
   - 強み: ドメインパワー高、記事数多い
   - 弱み: 東南アジア特化ではない

2. **海外転職サイトB**
   - 強み: 求人数多い
   - 弱み: ブログコンテンツ少ない

3. **現地情報サイトC**
   - 強み: 生活情報充実
   - 弱み: 求人への導線なし

### 差別化ポイント
- **東南アジア特化の専門性**
- **求人 + 生活情報の一体化**
- **現地スタッフによるリアルな情報**
- **20年以上の実績と信頼性**
