**最終更新: 2026-01-20**

---

## 目次について

### 重要：手動目次は不要

WordPressには目次自動生成プラグインがインストールされています。そのため、**記事内に手動で目次を追加する必要はありません**。

手動で目次を追加すると、プラグインが生成する目次と重複して2つの目次が表示されてしまいます。

### NG例（追加しないでください）

```html
<!-- これは追加しないでください -->
<h2>目次</h2>
<ul>
<li>セクション1</li>
<li>セクション2</li>
</ul>

<!-- これも追加しないでください -->
<nav>
<h4>目次</h4>
<ol>
<li><a href="#section1">セクション1</a></li>
<li><a href="#section2">セクション2</a></li>
</ol>
</nav>
```

### OK例

見出し（h2, h3）を適切に使用すれば、プラグインが自動的に目次を生成します。

```html
<p>導入文...</p>

<h2>セクション1</h2>
<p>内容...</p>

<h3>サブセクション1-1</h3>
<p>詳細...</p>

<h2>セクション2</h2>
<p>内容...</p>
```

---

## 見出しの使い方

### 見出しレベル

- `<h2>` : メインセクション（目次に表示される）
- `<h3>` : サブセクション（目次に表示される）
- `<h4>` : 小見出し（目次に表示されない場合あり）

### 見出しにIDを付ける

ページ内リンク用にIDを付けることができます（任意）。

```html
<h2 id="salary">給与・年収</h2>
```

## 記事構成の基本

### 推奨構成

1. **導入文** - 記事の概要、対象読者、読むメリット
2. **本記事のポイント（要約）** - 箇条書きで主要ポイント
3. **本文** - h2/h3で構造化
4. **FAQ** - よくある質問
5. **まとめ** - 結論、CTA
6. **関連記事** - 内部リンク

### 文字数目安

- 最低: 20,000文字
- 理想: 25,000〜35,000文字
- 専門記事: 40,000文字以上も可

## 対象読者（ターゲット）の明確化（重要）

### ルール

記事作成時は必ず**対象読者を明確にし、インデックスに記載**すること。

### 対象読者の分類

| 分類 | 説明 | タイトル例 |
|-----|------|-----------|
| **求職者** | 転職・就職希望者 | 「〇〇転職ガイド」「〇〇で働く」「ビザ取得方法」 |
| **企業・経営者** | 進出企業、投資家 | 「〇〇投資ガイド」「外資規制」「BOI恩典」 |
| **企業・人事** | 人事・採用担当者 | 「労働法ガイド（経営者向け）」「雇用規制」 |
| **全般** | 特定しない | 「最低賃金」「経済動向」 |

### タイトルの付け方

対象読者に合わせたタイトルにする：

**求職者向け：ポジティブな表現**
- ✅ 「タイ就労ビザ・ワークパーミット完全ガイド」
- ❌ 「タイ外国人雇用規制ガイド」（堅苦しい）

**企業向け：専門的でも可**
- ✅ 「タイ労働法ガイド【経営者・人事向け】」
- ✅ 「ベトナム外資規制・投資法ガイド」

### インデックス記載

記事公開時は `記事インデックス.md` の該当テーブルに「対象読者」を必ず記載する。

```markdown
| ID | URL | タイトル | 対象読者 | 文字数 | 状態 |
|----|-----|---------|---------|-------|------|
| 8005 | thailand-visa-work-permit-guide-2026 | タイ就労ビザ・ワークパーミット完全ガイド | 求職者 | 18,823字 | ✅ 公開中 |
```

## CTAリンク先のルール（重要）

### ターゲット別のリンク先

記事のターゲット読者によって、CTAのリンク先を使い分けてください。

| ターゲット | リンク先 | URL |
|-----------|---------|-----|
| **求職者向け** | 求職者登録 | `https://kyujin.careerlink.asia/jobseeker/register` |
| **企業・経営者向け** | お問い合わせ | `https://kyujin.careerlink.asia/お問い合わせ` |

### 求職者向け記事の例

- 転職ガイド（タイ転職、ベトナム転職など）
- 職種別ガイド（エンジニア、営業など）
- 年代別ガイド（新卒、40代、50代など）
- 生活費ガイド
- 給与・年収ガイド

```html
<p><a href="https://kyujin.careerlink.asia/jobseeker/register">転職相談はこちら</a></p>
```

### 企業・経営者向け記事の例

- 労働法ガイド（経営者・人事向け）
- 外国人雇用規制ガイド
- 投資法・外資規制ガイド
- BOI投資恩典ガイド
- 最低賃金ガイド
- 日系企業動向
- 経済動向・成長予測

```html
<p><a href="https://kyujin.careerlink.asia/お問い合わせ">お問い合わせはこちら</a></p>
```

### 判断基準

- **「〇〇への転職」「〇〇で働く」** → 求職者向け → 求職者登録
- **「経営者向け」「人事向け」「投資」「進出」** → 企業向け → お問い合わせ

## 禁止事項

### 絶対に避けること

1. **日系企業の批判** - クライアント企業のため批判禁止
2. **手動目次の追加** - プラグインと重複する
3. **根拠のない情報** - 特に数字や法律関連
4. **古い情報** - 2026年最新の情報を使用

### 注意が必要なこと

1. **給与情報** - 幅を持たせる（例：「300〜500万円」）
2. **法律情報** - 「最新情報は各機関にご確認ください」と注記
3. **成功事例** - 具体的すぎる個人情報は避ける

## URLスラッグのルール

### 基本ルール

- **英語/ローマ字で短く**（日本語タイトルをそのまま使わない）
- タイトル全文を入れず、主要キーワードのみ

### 形式

```
[国]-[テーマ]-[年度（必要に応じて）]
```

### 良い例・悪い例

| ❌ 悪い例 | ✅ 良い例 |
|----------|----------|
| ベトナム外資規制・投資法完全ガイド2026【業種別... | vietnam-foreign-investment-regulations-2026 |
| タイBOI投資恩典完全ガイド2026【法人税免除... | thailand-boi-investment-incentives-2026 |
| タイ就労ビザ・ワークパーミット完全ガイド | thailand-visa-work-permit-guide-2026 |

### 注意

日本語がURLエンコードされると `%e3%83%99%e3%83%88...` のような長い文字列になり、SEOにも悪影響。
