# メタディスクリプション最適化 - コピペで使える

## WordPressでの設定方法
1. Rank Math または Yoast SEO プラグインをインストール
2. 各ページの編集画面でメタディスクリプションを入力
3. または、テーマのheader.phpに直接追加

## OGP設定（SNSシェア用）

各ページに以下も追加することを推奨：

```html
<!-- OGP設定 -->
<meta property="og:title" content="[ページタイトル]">
<meta property="og:description" content="[メタディスクリプションと同じ]">
<meta property="og:type" content="website">
<meta property="og:url" content="[ページURL]">
<meta property="og:image" content="https://kyujin.careerlink.asia/ogp-image.jpg">
<meta property="og:site_name" content="キャリアリンクアジア">
<meta property="og:locale" content="ja_JP">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="[ページタイトル]">
<meta name="twitter:description" content="[メタディスクリプション]">
<meta name="twitter:image" content="https://kyujin.careerlink.asia/ogp-image.jpg">
```

## Rank Math での設定手順

1. WordPress管理画面 → Rank Math → タイトルと説明
2. ホームページ設定でトップページのタイトル・説明を入力
3. 投稿タイプ → 求人ページのテンプレートを設定
4. タクソノミー → カテゴリー・タグページの設定

## Yoast SEO での設定手順

1. WordPress管理画面 → Yoast SEO → 検索での見え方
2. 一般タブでトップページの設定
3. コンテンツタイプで各ページタイプのテンプレート設定
4. 個別ページは編集画面下部のYoastセクションで設定
