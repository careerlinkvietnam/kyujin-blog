# 技能実習制度とは？2027年「育成就労制度」への移行を徹底解説【2026年最新版】

**最終更新：2026年1月**

外国人技能実習制度は、2027年4月1日より「育成就労制度」へ移行することが決定しました。本記事では、現行の技能実習制度の概要と、新制度への移行について詳しく解説します。

---

## 技能実習制度とは

技能実習制度は、開発途上国等の外国人を日本に受け入れ、日本の技術・技能・知識を習得させ、母国の経済発展に貢献する「人づくり」を目的とした制度です。

### 法的根拠

- **技能実習法**（正式名称：外国人の技能実習の適正な実施及び技能実習生の保護に関する法律）
- 公布：2016年11月28日
- 施行：2017年11月1日

### 制度の基本理念

> 「技能実習は、労働力の需給の調整の手段として行われてはならない」

この基本理念のもと、技能実習生の人権保護と適正な実習の実施が求められています。

---

## 技能実習の区分と在留資格

| 区分 | 在留資格 | 期間 | 要件 |
|------|---------|------|------|
| 第1号技能実習 | 技能実習1号イ・ロ | 1年目 | 入国時の講習修了 |
| 第2号技能実習 | 技能実習2号イ・ロ | 2〜3年目 | 技能検定基礎級合格 |
| 第3号技能実習 | 技能実習3号イ・ロ | 4〜5年目 | 技能検定3級合格 + 優良認定 |

**通算最長5年**の在留が可能です。

### 企業単独型と団体監理型

- **企業単独型（イ）**：日本企業が海外の現地法人・合弁企業等から直接受入れ
- **団体監理型（ロ）**：監理団体が受入れ、傘下の企業で実習を実施（約98%がこの形態）

---

## 対象職種・作業

現行の技能実習制度では、**90職種165作業**が対象です。

### 主な職種カテゴリー

| 分野 | 職種例 |
|------|--------|
| 農業関係 | 耕種農業、畜産農業 |
| 漁業関係 | 漁船漁業、養殖業 |
| 建設関係 | 建築大工、左官、とび |
| 食品製造関係 | 缶詰巻締、食鳥処理加工業 |
| 繊維・衣服関係 | 紡績運転、織布運転 |
| 機械・金属関係 | 鋳造、鍛造、機械加工 |
| その他 | 介護、ビルクリーニング |

---

## 技能実習制度の問題点

技能実習制度は、以下の問題点が指摘されてきました：

### 1. 制度目的と実態の乖離

- 名目上は「技能移転」だが、実質的には「労働力確保」として活用
- 帰国後の技能活用の実態が不透明

### 2. 人権問題

- 長時間労働・低賃金
- パスポート・在留カードの取り上げ
- 劣悪な住環境

### 3. 転籍制限

- 原則として実習先の変更が認められない
- 不当な扱いを受けても逃げ場がない

### 4. 失踪問題

- 劣悪な環境から逃れるための「失踪」が多発
- 不法滞在者の増加

これらの問題を解決するため、新たに「育成就労制度」が創設されました。

---

## 育成就労制度とは（2027年4月1日施行）

### 施行スケジュール

| 時期 | 内容 |
|------|------|
| 2024年6月 | 改正入管法成立 |
| 2026年度 | 監理支援機関の許可申請受付開始 |
| **2027年4月1日** | **育成就労制度 施行** |
| 〜2030年頃 | 移行期間（両制度が併存） |

### 制度の目的

> 「日本の人手不足分野における人材の育成・確保」

技能実習制度の「国際貢献」から、**人材確保**を正面から目的とする制度へ転換します。

---

## 技能実習制度と育成就労制度の比較

| 項目 | 技能実習制度 | 育成就労制度（2027年〜） |
|------|-------------|----------------------|
| 目的 | 技能移転・国際貢献 | **人材育成・確保** |
| 在留期間 | 最長5年 | **原則3年** |
| 転籍（転職） | 原則不可 | **条件付きで可能** |
| 対象職種 | 90職種165作業 | **17分野**（特定技能19分野から航空・自動車運送業を除く） |
| 日本語要件 | なし | **N5相当以上**（入国前） |
| 前職要件 | あり | **撤廃** |
| 特定技能との関係 | 移行可能 | **特定技能1号への移行を前提** |
| 監理団体 | 監理団体 | **監理支援機関** |

### 転籍（転職）の条件

育成就労制度では、以下の条件を満たせば本人の意思で転籍が可能です：

1. 同一機関での就労が**1年を超えていること**（分野により1〜2年）
2. 技能・日本語能力の一定水準の達成
3. 転籍先の確保

ただし、地方から都市部への人材流出を防ぐため、分野ごとに転籍制限を設ける場合があります。

### 日本語能力要件

| 時期 | 要件 |
|------|------|
| 入国前 | **N5相当以上**の試験合格、または認定日本語教育機関での講習修了 |
| 入国後 | 継続的な日本語学習支援 |

---

## 移行期間の取り扱い

### 2027年4月1日時点で在留中の技能実習生

- 現行の在留資格のまま実習を継続可能
- 在留期間満了時に育成就労または特定技能へ移行

### 2027年4月1日以降の新規受入れ

- 原則として「育成就労」の在留資格で受入れ
- 技能実習の新規受入れは段階的に終了

### 併存期間

概ね**2030年頃まで**は、技能実習生と育成就労外国人が混在する移行期間となります。

---

## 受入れ機関・監理団体への影響

### 監理団体 → 監理支援機関

| 項目 | 変更点 |
|------|--------|
| 名称 | 監理団体 → **監理支援機関** |
| 許可要件 | 厳格化（外部監査の義務化等） |
| 業務内容 | 監理業務 + **キャリア支援**の強化 |

### 受入れ企業

- 育成就労計画の作成・認定申請
- 日本語学習支援の実施
- 転籍時の適切な対応

---

## 特定技能制度との連携

育成就労制度の最大の特徴は、**特定技能1号への移行を前提**としていることです。

### キャリアパス

```
育成就労（3年）→ 特定技能1号（5年）→ 特定技能2号（無期限）
```

### 育成就労から特定技能1号への移行要件

1. 3年間の育成就労を修了
2. 技能試験に合格
3. 日本語能力試験N4以上に合格

---

## 企業が今からすべき準備

### 2025年〜2026年

1. **情報収集**：出入国在留管理庁の発表を注視
2. **現行制度の見直し**：技能実習生の受入れ状況を確認
3. **特定技能への切り替え検討**：技能実習2号修了者の移行

### 2026年〜2027年

1. **監理支援機関の選定**：新制度に対応した機関との連携
2. **育成就労計画の準備**：認定申請に向けた書類作成
3. **日本語学習支援体制の構築**

---

## まとめ

- 技能実習制度は**2027年4月1日**に「育成就労制度」へ移行
- 移行期間（〜2030年頃）は両制度が併存
- 新制度では**転籍可能**・**日本語要件あり**・**特定技能への移行前提**
- 企業は今から情報収集と準備を進めることが重要
