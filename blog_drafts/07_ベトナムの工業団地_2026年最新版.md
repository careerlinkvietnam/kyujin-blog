# ベトナム工業団地ガイド【2026年最新版】｜概要・選び方・地域別解説

## 2024年〜2025年の主要トピックス

### 1. 住友商事：タインホア省で新規開発

| 項目 | 内容 |
|------|------|
| 名称 | THANG LONG INDUSTRIAL PARK THANH HOA（TLIP4） |
| 場所 | タインホア省（ハノイから南へ約150km） |
| 開発面積 | 第1期：約167ヘクタール |
| 総事業費 | 約170億円 |
| 着工 | 2025年秋 |
| 入居開始 | 2026年末頃予定 |

住友商事のベトナム5ヵ所目の工業団地。北中部エリアで日系企業が初めて開発する工業団地となる。

### 2. 丸紅：クアンニン省アマタシティ・ハロン

| 項目 | 内容 |
|------|------|
| 場所 | クアンニン省（中国国境近く） |
| 総開発面積 | 714ヘクタール |
| 丸紅出資比率 | 20%（2024年6月取得） |
| 主要入居企業 | Jinko Solar、Foxconn、多摩川精機など |

「チャイナ・プラスワン」の受け皿として、中国からのサプライチェーン移転需要に対応。

### 3. ロンタイン国際空港（2026年開港）

| マイルストーン | 日程 |
|--------------|------|
| 初の技術飛行 | 2025年12月19日（実施済み） |
| 商業運用開始 | **2026年6月予定** |
| 年間旅客能力（第1期） | 2,500万人 |

ホーチミン発着の全国際線が段階的に移管予定。ドンナイ省・ロンタイン周辺の工業団地に注目。

## 工業団地選定のポイント

### 1. 立地・アクセス

| チェックポイント | 解説 |
|-----------------|------|
| 主要都市からの距離 | ハノイ/ホーチミンから1〜2時間圏内が理想 |
| 港湾アクセス | 輸出入が多い場合は港近くが有利 |
| 空港アクセス | 出張者・駐在員の利便性 |
| 高速道路接続 | 物流コスト削減 |

### 2. インフラ

| チェックポイント | 解説 |
|-----------------|------|
| 電力供給 | 安定性、停電頻度、自家発電設備 |
| 用水 | 工業用水の供給量、水質 |
| 排水処理 | 環境基準への適合、処理費用 |
| 通信 | インターネット回線、光ファイバー |

### 3. コスト目安（2025年）

| 項目 | 北部（参考） | 南部（参考） |
|------|-------------|-------------|
| 土地リース料 | 80〜150 USD/m²/50年 | 100〜200 USD/m²/50年 |
| 電力単価 | 約7〜8セント/kWh | 約7〜8セント/kWh |
| 工場レンタル | 4〜6 USD/m²/月 | 5〜8 USD/m²/月 |

> ※料金は工業団地・条件により大きく異なります

### 4. 労働力

| チェックポイント | 解説 |
|-----------------|------|
| 労働力人口 | 周辺地域の人口、通勤圏 |
| 賃金水準 | 最低賃金（地域区分による） |
| 技術者確保 | 近隣の工業大学・職業訓練校 |
| 労働者の定着率 | 過去の離職率データ |
