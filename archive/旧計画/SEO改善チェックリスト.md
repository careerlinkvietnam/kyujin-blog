## 技術的SEO

### メタタグ最適化

#### タイトルタグ
- [ ] 各ページに固有のタイトルタグを設定（30-60文字）
- [ ] 主要キーワードを前方に配置
- [ ] ブランド名を末尾に統一

**推奨フォーマット:**
```
[主要キーワード] | [補足情報] | キャリアリンクアジア
```

**例:**
- トップ: `海外転職・東南アジア求人情報 | タイ・ベトナム就職 | キャリアリンクアジア`
- タイ求人: `タイ求人・バンコク就職情報【日系企業】| キャリアリンクアジア`
- ベトナム求人: `ベトナム求人・ハノイ/ホーチミン就職【日系企業】| キャリアリンクアジア`

#### メタディスクリプション
- [ ] 各ページに120-160文字のユニークな説明文
- [ ] CTAを含める（「今すぐ応募」「詳細を見る」等）
- [ ] 主要キーワードを自然に含める

**トップページ推奨例:**
```
タイ・ベトナム・東南アジアの日系企業求人を専門にご紹介。20年以上の実績で、あなたの海外転職をサポート。営業・製造・ITなど多数の求人掲載中。無料会員登録で非公開求人も閲覧可能。
```

### 見出し構造（Hタグ）
- [ ] 各ページにH1タグは1つのみ
- [ ] H2→H3→H4の階層構造を守る
- [ ] 見出しにキーワードを含める

### URL構造
- [ ] 日本語URLを避け、英語またはローマ字に統一
- [ ] URL階層を浅く保つ（3階層以内推奨）
- [ ] URLにキーワードを含める

**良い例:**
```
/vietnam/hanoi-jobs/
/thailand/bangkok-jobs/
/blog/vietnam-living-guide/
```

## 構造化データ（Schema.org）

### 求人ページ必須
```json
{
  "@context": "https://schema.org",
  "@type": "JobPosting",
  "title": "営業マネージャー",
  "description": "日系企業でのBtoB営業...",
  "datePosted": "2026-01-15",
  "validThrough": "2026-03-15",
  "employmentType": "FULL_TIME",
  "hiringOrganization": {
    "@type": "Organization",
    "name": "キャリアリンクアジア",
    "sameAs": "https://kyujin.careerlink.asia"
  },
  "jobLocation": {
    "@type": "Place",
    "address": {
      "@type": "PostalAddress",
      "addressLocality": "Bangkok",
      "addressCountry": "TH"
    }
  },
  "baseSalary": {
    "@type": "MonetaryAmount",
    "currency": "THB",
    "value": {
      "@type": "QuantitativeValue",
      "minValue": 70000,
      "maxValue": 100000,
      "unitText": "MONTH"
    }
  }
}
```
