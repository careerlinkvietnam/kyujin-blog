## 1. カテゴリー構造概要

## 4. カテゴリー割り当ての基本原則

1. **国・地域カテゴリーは必須**：記事が特定の国について書かれている場合、必ずその国カテゴリーを割り当てる
2. **都市サブカテゴリーは任意**：都市に特化した記事の場合、都市カテゴリーも追加する
3. **トピックカテゴリーは内容に応じて**：ビジネス、生活情報、グルメなど、記事の主題に合ったカテゴリーを割り当てる
4. **複数カテゴリーOK**：1つの記事に複数のカテゴリーを割り当て可能（最大4-5個が目安）
5. **サブカテゴリーがある場合**：サブカテゴリーを割り当てる場合、親カテゴリーも一緒に割り当てる

## 5. カテゴリー割り当て方法（API）

### Node.jsでカテゴリーを割り当てる例

```javascript
const https = require('https');

const AUTH = 'Basic ' + Buffer.from('careerlinkasia:N2Zz bzSn AWve Pa83 Ap2S 6Mlw').toString('base64');

async function assignCategories(articleId, categoryIds) {
  const updateData = JSON.stringify({
    categories: categoryIds
  });

  const options = {
    hostname: 'kyujin.careerlink.asia',
    path: `/blog/wp-json/wp/v2/posts/${articleId}`,
    method: 'PUT',
    headers: {
      'Authorization': AUTH,
      'Content-Type': 'application/json',
      'Content-Length': Buffer.byteLength(updateData)
    }
  };

  // ... リクエスト実行
}

// 例：ベトナム労働法ガイドにカテゴリーを割り当て
assignCategories(7998, [25, 148, 158, 273]);
// 25=ベトナム, 148=ビジネス, 158=法律, 273=採用・雇用
```

## 7. 注意事項

- **社労士は削除**：ベトナム・タイには日本の「社会保険労務士」に相当する資格はない。専門家として記載する場合は「弁護士」「会計事務所」「人事コンサルタント」を使用する
- **カテゴリー変更時は既存記事も確認**：新しいカテゴリールールを適用する際は、既存記事の整合性も確認する
- **未分類は避ける**：すべての記事に適切なカテゴリーを割り当てる
