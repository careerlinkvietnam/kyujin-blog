# 構造化データ（Schema.org）- コピペで使えるコード

## 概要
構造化データを設定すると、Googleの検索結果にリッチリザルトとして表示されます。
特に**JobPosting**は「Google しごと検索」に求人が表示されるため必須です。

---

## 3. WebSite（サイト全体）

### トップページに設置（サイト内検索ボックス表示用）
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "キャリアリンクアジア",
  "alternateName": "CareerLink Asia",
  "url": "https://kyujin.careerlink.asia",
  "potentialAction": {
    "@type": "SearchAction",
    "target": {
      "@type": "EntryPoint",
      "urlTemplate": "https://kyujin.careerlink.asia/search?q={search_term_string}"
    },
    "query-input": "required name=search_term_string"
  }
}
</script>
```

## WordPressでの設置方法

### 方法1: プラグインを使う（推奨）
1. **Rank Math** をインストール
2. 設定 → スキーママークアップで自動生成を有効化
3. 求人投稿タイプにJobPostingスキーマを適用

### 方法2: テーマに直接追加
1. WordPress管理画面 → 外観 → テーマエディター
2. header.php の `</head>` 直前に追加
3. または single.php の求人テンプレートに追加

### 方法3: functions.php で追加
```php
// 求人詳細ページに構造化データを追加
function add_job_posting_schema() {
    if (is_singular('job')) { // 求人の投稿タイプ名に合わせて変更
        $job_title = get_the_title();
        $job_description = get_the_content();
        // ... その他のフィールドを取得

        $schema = array(
            '@context' => 'https://schema.org/',
            '@type' => 'JobPosting',
            'title' => $job_title,
            // ...
        );

        echo '<script type="application/ld+json">' . json_encode($schema, JSON_UNESCAPED_UNICODE) . '</script>';
    }
}
add_action('wp_head', 'add_job_posting_schema');
```
