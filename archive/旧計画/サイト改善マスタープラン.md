# キャリアリンクアジア サイト改善マスタープラン
## 目標: アクセス数100倍達成

---

## 現状分析

### 強み
- 東南アジア7カ国をカバーする専門性
- 日系企業に特化したニッチなポジショニング
- ブログ「暮らすアジア」で実用的なコンテンツを提供
- Schema.orgマークアップなど基本的なSEO対策済み
- 定期的なコンテンツ更新

### 改善が必要な点
- メタディスクリプションの最適化
- 求人数が約25件と少ない
- ページ読み込み速度（JavaScriptが多い）
- バックリンク戦略
- SNSマーケティングの強化

---

## 100倍達成のための5つの柱

### 1. SEO強化（検索流入を10倍に）

#### 技術的SEO
- [ ] 全ページのメタディスクリプション最適化
- [ ] Core Web Vitals改善（LCP, FID, CLS）
- [ ] モバイルファーストインデックス対応確認
- [ ] サイトマップの最適化
- [ ] 内部リンク構造の改善
- [ ] 構造化データ（JobPosting Schema）の実装強化

#### コンテンツSEO
- [ ] ロングテールキーワード戦略
- [ ] 「海外転職」「タイ 就職」「ベトナム 日系企業」などの主要キーワードで上位表示
- [ ] FAQ形式のコンテンツ追加
- [ ] 各国別の詳細な就職ガイドページ作成

### 2. コンテンツマーケティング（ブログ流入を20倍に）

#### 記事戦略
- **毎週3本以上**の新規記事投稿
- ターゲットキーワードに基づいた計画的な記事作成
- 以下のカテゴリーを強化:
  1. 海外転職体験談・インタビュー
  2. 各国の生活費・給与相場比較
  3. ビザ・労働許可証の取得ガイド
  4. 現地での日本人コミュニティ情報
  5. リモートワーク×東南アジア拠点

#### 人気記事の最適化
- 既存の人気記事をリライト・更新
- 内部リンクの追加
- CTAの改善（求人への誘導）

### 3. SNS・ソーシャルマーケティング（新規流入経路を5倍に）

#### プラットフォーム別戦略

**Facebook**
- 海外在住日本人向けグループへの参加・情報発信
- 週3回以上の投稿
- 求人情報のシェア

**X（Twitter）**
- 海外転職・海外就職に関するツイート
- 業界ニュース・トレンドの発信
- ハッシュタグ戦略: #海外転職 #タイ就職 #ベトナム転職 #日系企業

**LinkedIn**
- B2B向け情報発信
- 採用企業への認知拡大
- 人材紹介サービスのPR

**YouTube（新規）**
- 海外転職ガイド動画
- 現地生活紹介
- 転職成功者インタビュー

**LINE公式アカウント強化**
- セグメント配信
- 求人アラート機能

### 4. 被リンク獲得戦略（ドメインパワーを3倍に）

#### 施策
- [ ] ゲスト投稿（海外転職系メディア）
- [ ] プレスリリース配信（PR TIMES等）
- [ ] 日本商工会議所との連携
- [ ] 現地日本人会への情報提供
- [ ] 業界調査レポートの公開（被リンク誘発）
- [ ] 他の人材紹介サイトとの相互リンク

### 5. コンバージョン最適化（CVRを2倍に）

#### 求職者向け
- [ ] 会員登録フォームの簡素化
- [ ] LINE連携でのワンクリック登録
- [ ] 求人アラートメールの改善
- [ ] チャットボット導入

#### 採用企業向け
- [ ] 成功事例ページの充実
- [ ] 料金体系の明確化
- [ ] 問い合わせフォームの改善

## 具体的なアクションプラン

### Phase 1: 基盤整備（1-2ヶ月目）
1. SEO監査と技術的問題の修正
2. キーワードリサーチと記事計画策定
3. SNSアカウント整備・運用ルール策定
4. Google Search Console / Analytics の詳細分析

### Phase 2: コンテンツ拡充（3-4ヶ月目）
1. 週3本以上のブログ記事投稿開始
2. 国別就職ガイドページ作成
3. SNS定期投稿開始
4. メールマガジン強化

### Phase 3: 拡散・認知拡大（5-6ヶ月目）
1. ゲスト投稿・被リンク獲得活動
2. YouTube動画コンテンツ開始
3. 広告運用（Google Ads, Facebook Ads）
4. インフルエンサーマーケティング

### Phase 4: 最適化・スケール（7-12ヶ月目）
1. データ分析に基づく改善
2. 成功施策の拡大
3. 新規市場（国）の開拓
4. オフラインイベント（セミナー等）
