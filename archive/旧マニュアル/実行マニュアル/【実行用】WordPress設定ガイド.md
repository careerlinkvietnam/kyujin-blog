# WordPress SEO設定ガイド
## 暮らすアジア ブログ用

## 必須プラグインと設定

### 1. Rank Math（SEOプラグイン）

#### インストール手順
1. WordPress管理画面 → プラグイン → 新規追加
2. 「Rank Math」で検索
3. インストール → 有効化
4. セットアップウィザードに従って初期設定

#### 推奨設定

**一般設定**
```
タイトル & メタ → ホームページ
・SEOタイトル: 暮らすアジア｜東南アジアの生活・転職・旅行情報ブログ
・メタディスクリプション: 東南アジアの"今"を伝える情報ブログ。タイ・ベトナム在住スタッフが生活費・ビザ・グルメ・観光など現地情報を発信。海外転職・移住を検討中の方必見。
```

**投稿のデフォルト設定**
```
タイトル & メタ → 投稿
・SEOタイトル: %title% | 暮らすアジア
・メタディスクリプション: %excerpt%
```

**スキーマ設定**
```
一般設定 → スキーママークアップ
・Organization を有効化
・会社名: キャリアリンクアジア
・ロゴ: サイトロゴをアップロード
```

**サイトマップ設定**
```
サイトマップ → 設定
・サイトマップを有効化
・投稿を含める: はい
・ページを含める: はい
・カテゴリーを含める: はい
```

---

### 2. WP Super Cache（キャッシュプラグイン）

#### インストール手順
1. プラグイン → 新規追加
2. 「WP Super Cache」で検索
3. インストール → 有効化

#### 推奨設定
```
設定 → WP Super Cache

簡単設定:
・キャッシング: ON

詳細設定:
・ページを圧縮: ON
・キャッシュリビルド: ON
・CDN サポート: 必要に応じて
```

---

### 3. Smush（画像圧縮プラグイン）

#### インストール手順
1. プラグイン → 新規追加
2. 「Smush」で検索
3. インストール → 有効化

#### 推奨設定
```
Smush → 設定

・自動圧縮: ON
・メタデータを削除: ON
・リサイズ: 最大幅 2048px
・Lazy Load: ON（重要！）
・WebP変換: ON（Pro版）
```

---

### 4. Broken Link Checker（リンク切れチェック）

#### インストール手順
1. プラグイン → 新規追加
2. 「Broken Link Checker」で検索
3. インストール → 有効化

#### 推奨設定
```
設定 → リンクチェッカー

・チェック間隔: 72時間
・メール通知: ON
```

---

### 5. Redirection（リダイレクト管理）

#### インストール手順
1. プラグイン → 新規追加
2. 「Redirection」で検索
3. インストール → 有効化

#### 使用方法
```
ツール → Redirection

旧URL → 新URL のリダイレクトを設定
（URLを変更した場合に使用）
```

## 記事投稿時のSEOチェックリスト

### 投稿前に確認すること

- [ ] **タイトル**（32-60文字、キーワードを含む）
- [ ] **スラッグ**（英語/ローマ字、短く）
- [ ] **メタディスクリプション**（120-160文字）
- [ ] **アイキャッチ画像**（1200x630px推奨）
- [ ] **見出し構造**（H2→H3→H4の階層）
- [ ] **内部リンク**（関連記事に3件以上）
- [ ] **外部リンク**（権威あるサイトに1件以上）
- [ ] **画像のalt属性**（すべての画像に設定）
- [ ] **カテゴリー**（1つのメインカテゴリー）
- [ ] **タグ**（3-5個程度）

### Rank Mathのスコア目標

- SEOスコア: **80点以上**を目指す
- 読みやすさスコア: **良好**以上

## 記事作成テンプレート

### タイトルの付け方

```
【年度】+ キーワード + 補足情報

例:
・【2026年版】タイ就職完全ガイド｜ビザ・給与・生活費まで徹底解説
・未経験から海外転職する方法｜成功する人の特徴と具体的なステップ
・タイ・バンコクの生活費｜日本人が暮らす実際の費用を公開
```

### 記事構成テンプレート

```markdown
# タイトル（H1）

導入文（200-300文字）
・読者の悩み・疑問に触れる
・この記事で解決できることを示す

## 目次

## 見出し1（H2）

本文...

### 小見出し1-1（H3）

本文...

### 小見出し1-2（H3）

本文...

## 見出し2（H2）

...

## まとめ（H2）
・要点を3-5点で整理
・次のアクションを提示

## CTA
・求人ページへのリンク
・会員登録への誘導

## 関連記事
・関連記事3件程度
```

## サイト速度改善

### PageSpeed Insights目標スコア

| 指標 | モバイル | PC |
|------|---------|-----|
| パフォーマンス | 70以上 | 80以上 |
| LCP | 2.5秒以下 | 2.5秒以下 |
| FID | 100ms以下 | 100ms以下 |
| CLS | 0.1以下 | 0.1以下 |

### 改善策

**画像最適化**
```
・WebP形式を使用
・適切なサイズにリサイズ
・Lazy Loadを有効化
```

**スクリプト最適化**
```
・不要なプラグインを削除
・JavaScriptの遅延読み込み
・CSSの最小化
```

**サーバー最適化**
```
・キャッシュプラグインを有効化
・CDN導入を検討（Cloudflare等）
・GZIP圧縮を確認
```

## Google Search Console 設定

### 設定手順

1. https://search.google.com/search-console にアクセス
2. 「プロパティを追加」をクリック
3. サイトURLを入力
4. 所有権を確認（HTML タグ方式推奨）

### Rank Mathとの連携

```
Rank Math → 一般設定 → ウェブマスターツール
・Google Search Console の認証コードを入力
```

### 確認すべき項目（週次）

- [ ] インデックスカバレッジ（エラーがないか）
- [ ] 検索パフォーマンス（クリック数、表示回数）
- [ ] Core Web Vitals
- [ ] 手動ペナルティの有無

---

## Google Analytics 4 設定

### 設定手順

1. https://analytics.google.com にアクセス
2. プロパティを作成
3. データストリームを追加（ウェブ）
4. 測定ID（G-XXXXXXXX）を取得

### WordPress への設置

**方法1: Rank Math で設置**
```
Rank Math → 一般設定 → Analytics
・Google Analytics 4 の測定IDを入力
```

**方法2: Site Kit プラグイン**
```
1. Site Kit by Google をインストール
2. Google アカウントと連携
3. 自動的に設置される
```

### 確認すべき指標（月次）

| 指標 | 確認ポイント |
|------|-------------|
| ユーザー数 | 増加傾向か |
| セッション数 | 増加傾向か |
| 平均エンゲージメント時間 | 3分以上か |
| 直帰率 | 60%以下か |
| 流入元 | オーガニック検索が増えているか |
