## 4. 計測イベント仕様

### 4.1 CTAクリック計測

| 項目 | 内容 |
|------|------|
| **イベント名** | `cta_click` |
| **発火条件** | `.cta-box a` または `a[href*="お問い合わせ"]` クリック時 |
| **パラメータ** | `article_id`, `country`, `article_type`, `cta_strength`, `cta_position`, `cta_text` |

### 4.2 フォーム到達計測

| 項目 | 内容 |
|------|------|
| **イベント名** | `form_view` |
| **発火条件** | `/お問い合わせ` ページビュー時 |
| **パラメータ** | `referrer_article_id`, `referrer_country`, `referrer_article_type`, `referrer_cta_strength` |

### 4.3 フォーム送信完了計測

| 項目 | 内容 |
|------|------|
| **イベント名** | `form_submit` |
| **発火条件** | フォーム送信成功時（サンクスページ到達 or submitイベント） |
| **パラメータ** | `referrer_article_id`, `referrer_country`, `referrer_article_type`, `referrer_cta_strength`, `inquiry_type` |

### 4.4 パラメータ値の例

| パラメータ | 型 | 値の例 |
|-----------|-----|--------|
| `article_id` | string | "6992", "8563" |
| `country` | string | "thailand", "vietnam", "general" |
| `article_type` | string | "専門性証明", "信頼構築", "集客", "補助" |
| `cta_strength` | string | "light", "standard", "strong" |
| `cta_position` | string | "hero", "middle", "bottom", "sidebar" |
