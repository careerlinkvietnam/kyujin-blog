# 育成就労制度とは？2027年4月施行の新制度を徹底解説

**最終更新：2026年1月**

2027年4月1日より、技能実習制度に代わる新たな外国人材受入れ制度「育成就労制度」が施行されます。本記事では、新制度の概要、技能実習制度との違い、企業が準備すべきことを詳しく解説します。

---

## 育成就労制度とは

「育成就労制度」は、日本の人手不足分野において、**外国人材を育成しながら確保する**ことを目的とした新しい在留資格制度です。

### 制度の背景

| 課題 | 内容 |
|------|------|
| 技能実習制度の問題点 | 人権問題、転籍制限、制度目的と実態の乖離 |
| 人手不足の深刻化 | 生産年齢人口の減少、特定産業の労働力不足 |
| 国際競争 | 外国人材獲得競争の激化（韓国、台湾等との競争） |

### 法的根拠

- **改正入管法**：2024年6月成立
- **施行日**：2027年4月1日
- **移行期間**：約3年間（〜2030年頃まで技能実習制度と併存）

---

## 制度の目的

### 技能実習制度との比較

| 項目 | 技能実習制度 | 育成就労制度 |
|------|-------------|-------------|
| 目的 | 技能移転・国際貢献 | **人材育成・確保** |
| 建前と実態 | 乖離あり | 一致 |

> **ポイント**：育成就労制度は、「人材確保」を正面から目的として認めた画期的な制度です。

---

## 育成就労制度の主な特徴

### 1. 特定技能への移行を前提

育成就労制度の最大の特徴は、**特定技能1号への移行を前提**とした設計です。

```
育成就労（3年）→ 特定技能1号（5年）→ 特定技能2号（無期限）
        ↓
   日本での長期就労・永住への道
```

### 2. 転籍（転職）が可能に

| 項目 | 技能実習制度 | 育成就労制度 |
|------|-------------|-------------|
| 本人意思による転籍 | 原則不可 | **条件付きで可能** |
| やむを得ない事情による転籍 | 制限あり | 認められる |

#### 転籍の条件

1. **同一機関での就労が1〜2年**（分野ごとに設定）
2. 技能検定試験基礎級等に合格
3. **日本語能力A1〜A2相当**に合格（分野ごとに設定）
4. 転籍先の確保（同一業務区分内に限る）

> **注意**：地方から都市部への人材流出を防ぐため、分野によっては転籍制限が設けられる場合があります。

### 3. 日本語能力要件の新設

| 時期 | 要件 |
|------|------|
| 入国前 | **N5相当以上**の試験合格、または認定日本語教育機関での講習修了 |
| 1年目終了時 | A1相当以上 |
| 3年目終了時 | A2相当以上（特定技能1号移行時はN4以上） |

### 4. 前職要件の撤廃

| 項目 | 技能実習制度 | 育成就労制度 |
|------|-------------|-------------|
| 前職要件 | あり（関連業務経験が必要） | **撤廃** |
| 復職要件 | あり（帰国後の技能活用） | **撤廃** |

---

## 対象分野

### 対象分野（17分野）

育成就労制度の対象分野は、特定技能制度の特定産業分野（19分野）から**航空**と**自動車運送業**を除いた**17分野**です。

| 分野 | 備考 |
|------|------|
| 介護 | 日本語要件がより厳格 |
| ビルクリーニング | - |
| 工業製品製造業 | 旧・素形材等3分野を統合 |
| 建設 | - |
| 造船・舶用工業 | - |
| 自動車整備 | - |
| 宿泊 | - |
| 農業 | - |
| 漁業 | - |
| 飲食料品製造業 | - |
| 外食業 | - |
| 鉄道 | 2024年追加 |
| 林業 | 2024年追加 |
| 木材産業 | 2024年追加 |
| リネンサプライ | 新規追加予定 |
| 物流倉庫 | 新規追加予定 |
| 資源循環 | 新規追加予定 |

> **注意**：技能実習制度の90職種165作業から大幅に縮小されます。航空・自動車運送業は育成就労の対象外です。

---

## 受入れの仕組み

### 監理団体 → 監理支援機関

| 項目 | 技能実習制度 | 育成就労制度 |
|------|-------------|-------------|
| 名称 | 監理団体 | **監理支援機関** |
| 許可要件 | - | 厳格化（外部監査義務化等） |
| 役割 | 監理 | 監理 + **キャリア支援** |

### 育成就労計画

受入れ企業は、外国人ごとに「育成就労計画」を作成し、認定を受ける必要があります。

#### 計画に含める内容

- 育成目標（3年後に特定技能1号水準に到達）
- 業務内容
- 日本語学習支援の計画
- 生活支援の計画

---

## 在留期間

| 区分 | 期間 |
|------|------|
| 育成就労（基本） | **原則3年** |
| 1回の在留期間 | 1年以内 |
| 更新 | 可能（計3年まで） |

### 特定技能への移行

3年間の育成就労を修了後、以下の要件を満たせば特定技能1号へ移行できます：

1. 技能検定試験3級等に合格
2. 日本語能力試験N4以上に合格

---

## 施行スケジュール

| 時期 | 内容 |
|------|------|
| 2024年6月 | 改正入管法成立 |
| 2025年〜2026年 | 関係省令・告示の整備 |
| **2026年度** | **監理支援機関の許可申請受付開始** |
| **2027年4月1日** | **育成就労制度 施行** |
| 2027年〜2030年頃 | 移行期間（技能実習制度と併存） |
| 2030年頃〜 | 育成就労制度に完全移行 |

## よくある質問（FAQ）

### Q1. 技能実習生を現在受入れていますが、どうなりますか？

**A1.** 2027年4月1日時点で在留中の技能実習生は、現行の在留資格のまま実習を継続できます。在留期間満了時に育成就労または特定技能へ移行することになります。

### Q2. 転籍で人材が流出しませんか？

**A2.** 転籍には「同一機関で1年以上就労」などの条件があり、すぐに転籍できるわけではありません。また、分野によっては地域間転籍に制限が設けられる予定です。良好な労働環境を整備することが人材定着の鍵となります。

### Q3. 日本語教育は誰が行いますか？

**A3.** 受入れ企業と監理支援機関が協力して行います。日本語学校との連携や、オンライン学習ツールの活用なども検討してください。

### Q4. 技能実習制度で受入れていた職種が対象外になる場合は？

**A4.** 育成就労制度の対象分野は17分野に限定されるため、技能実習で可能だった一部の職種は対象外となる可能性があります。今後の発表を注視してください。

## まとめ

| ポイント | 内容 |
|----------|------|
| 施行日 | 2027年4月1日 |
| 目的 | 人材育成・確保（技能移転から転換） |
| 在留期間 | 原則3年 |
| 転籍 | 条件付きで可能 |
| 日本語要件 | 入国前N5相当以上 |
| 対象分野 | 特定技能19分野から航空・運送業を除く17分野 |
| 特定技能との関係 | 1号への移行を前提 |

育成就労制度は、外国人材を「育てながら確保する」ことを目的とした新制度です。技能実習制度の問題点を解消しつつ、日本での長期就労への道を開くことで、より多くの優秀な外国人材を呼び込むことが期待されています。

企業は今から情報収集と準備を進め、2027年の施行に備えましょう。
