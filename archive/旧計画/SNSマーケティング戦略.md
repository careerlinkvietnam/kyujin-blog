## プラットフォーム別戦略

### X（Twitter）

#### アカウント設定
- **アカウント名:** キャリアリンクアジア｜東南アジア転職
- **プロフィール:**
```
タイ・ベトナム・東南アジアの海外転職を支援して20年以上
毎日、海外転職・現地生活の情報を発信中
無料で求人チェック → [リンク]
お気軽にDMください
```

#### 投稿カテゴリー
1. **求人紹介（30%）**
   - 新着求人の紹介
   - 人気求人のピックアップ

2. **ノウハウ・Tips（30%）**
   - 海外転職のコツ
   - 面接対策
   - 履歴書の書き方

3. **現地情報（20%）**
   - 生活費情報
   - 治安・生活の様子
   - 現地ニュース

4. **体験談・ストーリー（15%）**
   - 転職成功者の声
   - スタッフの日常

5. **エンゲージメント（5%）**
   - アンケート
   - 質問募集

#### 投稿頻度
- 1日2-3回
- 朝7-8時、昼12-13時、夜20-21時

#### 投稿テンプレート

**求人紹介:**
```
【新着求人】タイ・バンコク

職種: 営業マネージャー
給与: 70,000-100,000THB
条件: 営業経験3年以上

タイの日系製造業で営業チームを率いるポジション。
英語は日常会話レベルでOK

詳細はこちら→ [リンク]

#海外転職 #タイ就職 #バンコク求人
```

**ノウハウ系:**
```
【海外転職の面接でよく聞かれる質問5選】

1. なぜ海外で働きたいのか
2. 現地で何年働く予定か
3. 語学力について
4. 家族の理解は得られているか
5. 将来のキャリアプラン

回答のコツをスレッドで解説します

#海外転職 #転職活動
```
